Send calendar invites

Sends a personalized calendar invite (auto-adds to their calendar) to everyone with EVENT_PREF = OPTED_IN in Brevo.

ℹ️ How this works & what you need — read me

What this page does

It emails a personalized calendar invite to every CSCMP SoFlo member who opted in to calendar invites (Brevo attribute EVENT_PREF = OPTED_IN). Each person gets their own invite, so the event lands on their calendar automatically with Accept / Tentative / Decline — they don't have to import anything.

What you need before using it

  • Admin passcode — the value set in Netlify → Site configuration → Environment variables → ADMIN_SECRET.
  • An Eventbrite event — the invite details (title, date, venue) are pulled live from its Eventbrite page.
  • Members opted in — people who checked the calendar-invite box on the sign-up form (synced to Brevo as EVENT_PREF = OPTED_IN).
  • (Behind the scenes) the sender cscmp@sofloroundtable.org must be an authenticated sender in Brevo.

To send an invite

  1. Paste the Eventbrite event link and your passcode.
  2. Enter your own email and click Send test invite to my email — confirm it lands on your calendar.
  3. Click Send invites to ALL opted-in members. A progress bar shows how many were sent.

To change an event

Edit the event in Eventbrite, then come back and Send invites to ALL again. Because it's the same event, calendars update in place (no duplicate). Note: a change only reaches calendars when you re-send here — editing Eventbrite alone does nothing.

To cancel an event

Cancel it in Eventbrite first, then use Send CANCELLATION to ALL — it removes the event from everyone's calendars. (Test to yourself first with the cancellation test button.)

Good to know

  • Only opted-in, still-subscribed members receive it. People who unsubscribed in Brevo or clicked "stop calendar invites" are skipped automatically.
  • Each send uses your Brevo transactional email quota (about one send per opted-in member per event).
  • Always run the test to yourself before a full send.
Set once in Netlify → Environment variables → ADMIN_SECRET.

Cancel an event — sends a cancellation that removes it from members' calendars. (Cancel the event in Eventbrite first, then send this.)
